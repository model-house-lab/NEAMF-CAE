# NEAMF-CAE

# A CT-based Multi-scale Autoencoder with Transfer Learning for Distinguishing Multiple Primary Lung Cancers from Intrapulmonary Metastases

---

## 📦 Environment Setup

Make sure you have the following dependencies installed:

- Python >= 3.9  
- tensorflow==2.10
- numpy==1.26.4
- joblib==1.5.1
- pandas==2.3.0
- scikit-learn==1.7.0
- CUDA ≥ 11.2 (if using GPU)
- nrrd

Install dependencies with:

```bash
pip install -r requirements.txt
```
